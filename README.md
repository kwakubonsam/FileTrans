# FileTrans
This is a simple java desktop application sync files between clients and servers. 

# Getting Started 
You can clone the repo or download the files and run both the client and server files.

## Prerequisite to Run Code 
- Two computers 
- Any Java IDE 
- Internet connection

# Features
- TCP protocol
- Multi-threading
